Furnace Controller

This is a temperature controller unit, powered by a 230V AC supply. It features a digital display showing process ID, timer, and current cycle status, with buttons for operation, settings, and communication. The unit supports Wi-Fi connectivity for remote monitoring and data transmission. It communicates with the heating chamber via a dedicated temperature controller communication port for precise control.
